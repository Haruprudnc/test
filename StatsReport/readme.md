Check branches
```
get branch -a
```

Make new branch
```
git branch <branch_name>

```
git checkout -b <branch_name>
```

```
git checkout main
git branch -d <branch_name>
```

Switch branch
```
git checkout <branch_name>
```

Check status
```
git status
```

Merge branch
```
git merge <branch_name>
```

check how many clone
```
